Crea un script donde declares una función obtenerImporteConImpuestos que reciba dos
parámetros: el importe sin impuestos (numérico) y el tipo de producto (numérico entero).
La función debe mostrar por pantalla el importe sin impuestos más el 21% si el tipo de
producto es 1, ó el importe sin impuestos más el 10% si el tipo de producto es 2, ó el importe
sin impuestos más el 5% si el tipo de producto es 3. Si el usuario no especifica el tipo de
producto, consideramos que es de tipo 1. Devolver la cantidad con dos cifras decimales.
(Función declarada)
